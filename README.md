# Systems & AI Engineering Portfolio

Welcome! This repository is a collection of my projects in Artificial Intelligence, Machine Learning, and Data Engineering. These projects demonstrate my ability to handle end-to-end data pipelines, from raw data cleaning and statistical analysis to deploying Deep Learning models.

## 📁 Repository Structure

### [Computer Vision](./Computer_Vision/)
* **Pet Breed Classification:** A CNN-based model using TensorFlow and Keras to classify different pet species with image data augmentation.
* **Pet Emotion Detection:** Deep learning approach to identify animal emotional states from visual cues.

### [Natural Language Processing (NLP)](./Natural_Language_Processing/)
* **Amazon Review Sentiment Analysis:** Implementation of lemmatization and handling imbalanced classes to predict customer sentiment.
* **SMS Spam Detection:** A classification pipeline to filter spam messages using NLTK and Scikit-Learn.
* **IMDb Movie Sentiment:** Text vectorization and sentiment classification using TF-IDF.

### [Data Analysis & ML](./Data_Analysis/)
* **NYC 311 Service Analysis:** Extensive data engineering on public datasets, including time-series analysis and **Chi-Square Hypothesis Testing**.
* **IBM Employee Attrition:** Predicting staff turnover using HR analytics and classification algorithms.
